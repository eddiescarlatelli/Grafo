Programa feito para percorrer os dados de um grafo.
Foi programado em C++ e compilado usando o g++ do Linux.
Acomopanha o arquivo MakeFile para compilação.
Esse programa tem um menu que apresenta as opções de entrada que tem.
O programa pode apresentar o número de vértices e arestas do grafo, imprimir todos os vértices do grafo,
exibe qual o custo do caminho de menor distância entre dois vértices da escolha do usuário, mostra o diâmetro do grafo
e exibe qual vértice tem mais arestas (maior centralidade).
O programa acompanha o arquivo .txt de exemplo pra ter os dados necessários. Caso queira mudar os dados de entrada, é 
necessário criar um arquivo .txt com o nome de "exemplo", e salvar na mesma pasta que os arquivos a serem compilados.
Para executar o arquivo MakeFile, digite make no terminal do linux na localização dos arquivos e códigos fonte requeridos
pelo programa.
